# UCC IskolarEase: Web-Based Scholarship Management System

A web-based system designed to automate scholarship applications for students and scholarship providers.

**Frontend:** HTML5, CSS3, JavaScript  
**Backend:** PHP  
**Database:** MySQL (phpMyAdmin)  
**Tools:** XAMPP, Atom  

## 📌 Status
This project was developed as a capstone system and may contain incomplete or imperfect features. It is continuously being improved as part of ongoing learning and development.

## 🚀 Project Overview
UCC IskolarEase is a centralized web portal designed to digitize and automate the scholarship application process for incoming students. This project addresses the inefficiencies of manual, paper-based systems by providing a streamlined platform for students, university administrators, and scholarship providers.

### The Problem It Solves:
* **Manual Bottlenecks:** Replaces slow, physical document submission with digital uploads.
* **Lack of Transparency:** Provides real-time status tracking to eliminate student uncertainty.
* **Inefficient Verification:** Automates eligibility checks based on academic and financial criteria.

## ✨ Key Features

### For Students (Applicants)
* **Digital Application:** Apply for multiple scholarships through a single web portal.
* **Automated Eligibility Check:** System validation against scholarship requirements.
* **Real-time Notifications:** Receive updates on approval, rejection, or interview schedules.

### For Administrators & Providers
* **Scholarship Management:** Easily post new opportunities and manage deadlines.
* **Applicant Filtering:** Filter students by academic performance or financial need.
* **Reporting:** Generate basic statistics to support decision-making.

## 🛠️ Technical Implementation (CRUD & Logic)
* **Frontend:** Responsive UI built with HTML5, CSS3, and JavaScript.
* **Backend:** Server-side logic handled via PHP.
* **Database:** Structured MySQL database managing users, applications, and scholarship criteria.
* **Core Logic:** Implements filtering logic to match student profiles with eligible scholarships.

## ⚙️ Setup & Installation
1. **Clone the repository:**  
   `git clone https://github.com/paolorosalesdev/ucc-iskolarease.git`

2. **Database Setup:**  
   Create a database named `isko_db`, then import the provided `isko_db.sql` file using phpMyAdmin (XAMPP).

3. **Configuration:**  
   Update the database connection settings in your config file (host, username, password, database name).

4. **Launch:**  
   Access the system via `http://localhost/ucc-iskolarease`
